# Kata Calculator

Calculadora simple con operaciones básicas, construida con JavaScript (ESM) y testeada con Jest.

## Integrantes

- Chiesa, Máximo
- Goya, Santiago
- Iglina, Bruno
- Montini, Agostino

## Requisitos

- Node.js 22+
- pnpm

## Instalación

```bash
pnpm install
```

## Uso

```bash
node index.js
```

## Tests

```bash
pnpm test
```

En modo watch durante desarrollo:

```bash
pnpm test -- --watch
```
